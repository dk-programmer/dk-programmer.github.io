Who am I?

I'm a computer scientist based in Berlin, specializing in developing games and virtual worlds using Unity3D. My passion for game creation began over nine years ago, starting with a few prototype games. After a period of self-study, I entered the gaming industry as a part-time developer, crafting educational games in the forensic field. Currently, I'm working on the project 'UsUnited', where my primary responsibilities include feature implementation and bug fixing. If you're intrigued by my work or looking for collaboration, don't hesitate to reach out.

Street Light Simulation

×
Project Type Paid Work
Software Used Unity3D
Languages Used C#
Primary Role(s) Systemdesign, Programming

Details:

Real-time simulation of street lighting based on a provided image. This simulation is integrated with a database that hosts a wide selection of IES light profiles from various manufacturers, along with additional lighting data. Users can tailor these profiles to match a camera image, facilitating the simulation of potential street lighting scenarios. Developed using Unity, this simulation is a component of a Flutter application into which it has been integrated.

Contribution:

  • Conceptualization, and development of the system and integration
  • Feature implementing, testing and bug fixing
